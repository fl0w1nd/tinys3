# TinyS3 Changelog

## [1.0.0] - 2025-12-28

### Added

- **Upload Clipboard Image** - Upload images from clipboard to S3
- **Compress & Upload Clipboard Image** - Compress with TinyPNG before upload
- **Upload Selected Finder Images** - Batch upload images from Finder
- **Compress & Upload Selected Finder Images** - Batch compress and upload

### Features

- S3-compatible storage support (AWS S3, Cloudflare R2, MinIO, etc.)
- TinyPNG compression integration
- Multiple URL output formats (raw, Markdown, BBCode)
- Path style and Virtual-hosted style URL support
- Custom public URL base configuration
- Batch upload with progress indication
